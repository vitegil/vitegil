# Vitegil 🔭

### 安装

执行以下命令安装依赖

```bash
npm i
```
### 模拟mock环境下

执行以下命令启动项目

```bash
npm run dev-mock
```

### 开发环境下

执行以下命令启动项目，浏览器访问 http://localhost:5173

```bash
npm run dev
```
